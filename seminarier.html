Anmälan Seminarier

Fredagens seminarier

Modern webutveckling och React med Aurora Magnusson

En djupdykning i de senaste trenderna och teknikerna inom React och hur du kan använda dem för att skapa moderna användargränssnitt.

Optimering av webprestanda med Isidor Thorsen

Lär dig de bästa metoderna och verktygen för att optimera prestanda för webbplatser och appar, inklusive lasttider och användarupplevelse.

Responsiv webdesign i Praxis med Eleonora Viklund

Utforska bästa praxis för responsiv webbdesign och hur du skapar användarvänliga gränssnitt som fungerar på alla enheter.

Lördagens seminarier

Framtiden för CSS: CSS Grid och Flexbox med Maximilian Stenberg

En översikt över moderna layouttekniker med fokus på CSS Grid och Flexbox och hur de kan användas för att skapa flexibla och komplexa layouter.

Progressiva Web Appar (PWAs) med Selma Hjort

Upptäck fördelarna med PWAs och hur du utvecklar appar som fungerar offline, laddar snabbt och ger en app-liknande upplevelse i webbläsaren.

UI/UX Design för Frontend Developers med Elvin Åberg

Lär dig grunderna i användargränssnittsdesign och användarupplevelse och hur de påverkar frontend-utveckling.

Typsäkerhet i JavaScript med TypeScript med Juno Persson

En introduktion till TypeScript och hur det kan hjälpa dig att skriva säker och robust frontend-kod i JavaScript.

Söndagens seminarier

webbkomponenter och Component-Based Development med Leonora Holmström

Utforska användningen av webbkomponenter och fördelarna med komponentbaserad utveckling för att bygga återanvändbara och underhållbara gränssnitt.

Hantera Tillstånd i Frontend: Redux och Mobx med Valdemar Östergren

En översikt över tillståndsbehandling med Redux och Mobx, två populära tillvägagångssätt för att hantera tillstånd i frontend-applikationer.

Säkerhet i Frontend-utveckling med Vilda Lindqvist
